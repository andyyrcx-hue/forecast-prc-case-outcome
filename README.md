# forecast-prc-case-outcome

面向中国大陆法院诉讼研究的 Codex Skill：检索事实相似类案、权威裁判和承办法官公开裁判样本，分别评估事实相似度、权威参照级和法官样本强度，形成条件化裁判预测与诉讼策略。

> 本项目不是“算胜诉率”的工具，也不保证判决结果。它用于辅助律师更快看见裁判趋势、识别证据风险并组织诉讼论证，不能替代律师对事实、证据、法律效力和诉讼策略的最终判断。

## 能做什么

- 将案情拆成法律关系、争议焦点、决定性事实、证据结构和目标判项；
- 通过元典案例 MCP 检索权威案例、事实相似案例、反面案例和候选法官样本；
- 分开计算事实相似度 `S 0—100`、权威参照级 `A1—A6` 和法官样本强度 `J0—J4`；
- 有利案例强调与胜负开关有关的相同点，不利案例识别可能改变判项的真实差异；
- 比较上级法院裁判规则与承办法官公开裁判样本是否一致；
- 输出有利、基准、不利三种情景及补证、庭审、代理词和和解建议；
- 每次完成分析后同时生成 Markdown 主编辑版和 DOCX 正式阅读版。

## 适用范围

仅适用于中国大陆法院公开裁判文书研究。暂不适用于：

- 仲裁裁决；
- 其他国家或地区的司法预测；
- 未经授权的客户材料外部传输；
- 根据搜索摘要、模型记忆或单个法官样本作确定性结论。

## 安装

将 Skill 目录复制到 Codex Skills 目录：

```bash
cp -R skills/forecast-prc-case-outcome ~/.codex/skills/
```

随后新建一个 Codex 任务，或重新加载 Skills。

> 如果你的 `CODEX_HOME` 不在默认位置，请将目录复制到 `$CODEX_HOME/skills/`。

## 触发方式

最稳妥的显式触发：

```text
使用 $forecast-prc-case-outcome 分析这个案子。
```

也可以直接说：

- “查一下类似案例的判决倾向。”
- “类案预测：这个案子可能怎么判？”
- “查承办法官同类案件的裁判倾向。”
- “按最高法、高院、中院和基层法院分级检索类案。”

推荐输入模板：

```text
类案预测：我是原告/被告，案件由××法院审理，核心争议是……，
我方目标判项是……，现有关键证据是……。
请检索相似案例，按法院权威层级整理，分析承办法官公开裁判样本，
并预测本案可能怎么判。
```

## 输出成果

默认同时生成：

- `类案检索与裁判预测报告.md`：主编辑、持续更新版本；
- `类案检索与裁判预测报告.docx`：正式阅读版本；
- `检索记录与关键词.md`：脱敏检索式、数据源、检索日期、全文核验及排除记录。

报告包括：结论摘要、本案比较基准、权威轨、事实相似案例排序、承办法官轨、有利案例运用、不利案例区分、上下级观点冲突、三情景预测、行动建议及检索局限。

## 三套评分为什么分开

- `S` 回答“是不是类案”；
- `A` 回答“这个观点有多强的参照价值”；
- `J` 回答“能否说明承办法官存在相对稳定的公开裁判观察”。

三者不合成一个不透明的“胜诉指数”。有利结果不会提高相似度，法院级别也不会提高事实相似度。

## 隐私与安全

- 调用外部检索前，先删除或替换姓名、企业名、案号、合同编号、精确金额、联系方式、流水、证据原文和商业秘密；
- 法官姓名只有在用户明确要求检索该法官公开裁判并授权发送时，才作为外部查询条件；
- 不在仓库、示例、日志或检索记录中保存 API Key、访问令牌、MCP 配置秘密或真实客户信息；
- 无法取得全文时标注 `[摘要命中—待全文核验]`，不得把摘要包装成已核验裁判观点。
