# 评分方法

## 一、事实相似度 S（0—100）

| 维度 | 权重 | 评分问题 |
| --- | ---: | --- |
| 核心争议焦点 | 25% | 法院需要回答的法律问题是否同一 |
| 决定性事实 | 25% | 影响判项的胜负开关是否相同 |
| 法律关系 | 15% | 主体关系、请求权基础和责任结构是否相同 |
| 适用规范 | 15% | 法条、解释路径和时间效力是否相同 |
| 证据结构 | 10% | 证明责任、证据类型、缺口和争议状态是否相同 |
| 程序位置 | 5% | 一审/二审/再审、实体/程序问题是否可比 |
| 时空背景 | 5% | 法律版本、裁判时期、地域实践是否可比 |

`S = Σ(维度分 × 权重)`。

硬门槛：法律关系或核心争点低于 40 分时，原则上不得列为核心类案；决定性事实低于 40 分时，只能作为规则参考，不得称为高度相似。

- 85—100：高度相似；
- 70—84：较高相似；
- 55—69：部分相似；
- 40—54：仅具规则参考；
- 0—39：排除。

相似度不因判决对我方有利而上调，也不因法院级别高而上调。

## 二、权威参照级 A

| 级别 | 来源 |
| --- | --- |
| A1 | 最高人民法院指导性案例或对同一法律问题具有明确统一适用意义的权威文件/案例 |
| A2 | 最高人民法院生效裁判、公报案例、最高法发布的典型案例 |
| A3 | 省级高级人民法院生效裁判或正式发布的典型/参考案例 |
| A4 | 中级人民法院生效裁判、受理法院上级法院同类裁判 |
| A5 | 基层人民法院生效裁判、受理法院同类裁判 |
| A6 | 效力或全文状态待核验、程序裁定、法律版本明显不同，仅作线索 |

同级排序：事实相似度 → 同辖区/同法院 → 新近 → 全文与生效状态完整。不得把一般最高法个案表述为指导性案例。

## 三、法官样本强度 J

| 级别 | 标准 |
| --- | --- |
| J4 | 同一法官身份已核验，至少 5 件同争点可比样本，观点较一致且近期仍延续 |
| J3 | 同一法官身份已核验，3—4 件同争点或高度可比样本，方向较一致 |
| J2 | 同一法官身份已核验，2 件可比样本，或样本存在明显分化 |
| J1 | 仅 1 件、仅同案由不同争点、署名角色有限，或身份仍有局部疑问 |
| J0 | 未找到、重名无法排除、仅同法院而非同一法官，或文书未披露署名 |

“同一法官倾向”必须同时核验姓名、任职法院/时期、审判领域和文书署名。审判长、审判员、人民陪审员、法官助理、书记员的角色不可混同。合议庭裁判不能无条件归因于某一成员的个人观点。

## 四、倾向统计

按同一争点编码每案：支持我方规则、反对我方规则、中性/未处理。记录事实情境和理由，不只记录胜负。

仅当可比样本不少于 5 件时展示简单比例；分母是“全文核验且实际处理该争点的样本数”，不是搜索命中总数。样本少于 5 件时只作定性描述。

## 五、冲突矩阵

| 权威轨 | 法官轨 | 预测处理 | 代理表达 |
| --- | --- | --- | --- |
| 有利 | 有利 | 上调置信度 | 权威规则 + 本院一致实践 |
| 有利 | 不利 | 保留法官执行风险 | 权威规则为主，真实区分法官旧案 |
| 不利 | 有利 | 下调置信度并提示改判 | 不将个别旧案包装为稳定规则 |
| 不利 | 不利 | 作为核心风险 | 补证、限缩、备选请求；提示是否需要考虑和解，具体边界待用户确认 |
| 分化 | 样本不足 | 条件化预测 | 围绕胜负开关构建双向论证 |

## 六、置信度

- 高：核心事实证据明确，A1/A2 与高相似样本一致，法官轨无明显冲突；
- 中：存在可解释冲突、若干事实待核实，或只有 A3/A4 与一般样本支持；
- 低：关键事实未证实、权威观点分化、法官样本弱，或数据库覆盖/全文不足。
