# 虚构案情示例

> 本示例完全虚构，仅用于演示输入结构，不对应任何真实当事人、项目、案号、法官或裁判文书。

## 示例输入

```text
类案预测：我方为买方甲公司，拟在中国大陆法院起诉卖方乙公司。
双方签订工业设备买卖合同，约定设备完成安装调试并验收合格后支付尾款。
设备已交付并投入试运行，但双方没有签署正式验收单。
乙公司主张甲公司持续使用设备构成默示验收，要求支付全部尾款；
甲公司主张设备未达到约定产能，且曾通过邮件提出整改要求。

请检索相似案例，按法院权威层级整理，分析有利与不利类案，
并在公开文书样本充分时分析承办法官的裁判观察，预测本案可能怎么判。
```

## 建议补充的信息

- 我方诉讼地位与具体目标判项；
- 受理法院、程序阶段和已知承办法官；
- 验收条款原意的脱敏摘要；
- 试运行时间、整改通知和对方回复的证据状态；
- 是否存在付款、维修、退换货或质量鉴定材料。

## 检索前应展示的脱敏关键词示例

```text
工业设备买卖合同；安装调试；未签验收单；持续使用；默示验收；
产能不达标；邮件提出整改；尾款支付条件
```

未实际调用元典并核验全文前，具体案例、案号、裁判观点和法官倾向均应标注 `[待检索]`，不得根据本示例虚构。
